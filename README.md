# Landing Page de Alta Performance

## Esta é uma Landing Page de conversão desenvolvida para o nicho de Marketing Digital e Gestão de Tráfego. O projeto foi construído com foco em performance extrema, SEO e design persuasivo para transformar visitantes em leads qualificados.

🛠️ Tecnologias Utilizadas:

- Next.js 15 (App Router): Para renderização rápida e otimização de SEO.
- TypeScript: Garantindo um código robusto, tipado e de fácil manutenção.
- Tailwind CSS: Estilização moderna com foco em responsividade e performance visual.
- Lucide React: Biblioteca de ícones leves e consistentes.
- Web3Forms: Integração de formulário de contato serverless para recebimento real de e-mails.

✨ Diferenciais do Projeto:

- Design High-Ticket: Paleta de cores exclusiva (Gold & Emerald) focada em passar autoridade e exclusividade.
- Performance Otimizada: Estrutura leve para garantir carregamento instantâneo, essencial para campanhas de tráfego pago.
- Arquitetura Escalável: Conteúdo centralizado em arquivos de configuração (data.ts), permitindo trocar todo o texto da página em segundos.
- Segurança: Implementação de Variáveis de Ambiente (.env) para proteção de chaves de API em produção.

📦 Como rodar o projeto localmente:

1. Clone o repositório: git clone https://github.com/seu-usuario/lp-agencia-marketing
2. Instale as dependências: npm install
3. Configure sua NEXT_PUBLIC_WEB3FORMS_KEY no arquivo .env.local.
4. Inicie o servidor de desenvolvimento: npm run dev
